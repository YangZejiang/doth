# 后端Python项目结构(Flask)

``` txt
├── api                         // 路由控制器目录
├── config                      // 项目配置目录
├── model                       // 数据结构目录
├── static                      // 静态资源目录
├── utils                       // 工具类目录
│   ├── requestUtil.py          // 接口请求工具类
│   └── jsonUtil.py             // json字符操作解析工具类
├── main.py                     // 项目入口
└── README.md                   // 项目说明
```
